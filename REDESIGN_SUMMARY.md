## 💡 How to Use

### View the Portfolio
1. Main portfolio: Open `index.html`
2. Individual sites: Navigate to `portfolio/[site-name]/index.html`

### Add More Projects
1. Copy `fabrica-design-system.css` approach
2. Define brand colors in your CSS
3. Use the card/grid components
4. Follow the section structure

### Customize Colors
```css
:root {
    --primary: #your-primary-color;
    --secondary: #your-secondary-color;
    --accent: #your-accent-color;
}
```
