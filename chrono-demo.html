How Chrono Mode Works

Unlike traditional light/dark mode, Chrono Mode uses 8 distinct phases that flow naturally throughout the day

⏰ Time-Aware

Automatically detects the current time and applies the appropriate color phase. Works 24/7 without user intervention.

🎨 8 Distinct Phases

From dawn's soft pastels to night's deep blues, each phase is carefully designed to match natural lighting conditions.

🔄 Smooth Transitions

Colors fade gently between phases (2-3 seconds), creating a seamless, organic experience.

⚙️ Manual Override

Need to focus? Jump to any phase manually. Full control with automatic fallback.
