# Color Wheel Dial - Art of the Possible

## What It Is

An interactive color wheel that changes the entire page background as you drag around it.

## How It Works

1. **Open the file** - Open `color-wheel.html` in any modern browser
2. **Drag the wheel** - Click/tap anywhere on the donut chart and drag
3. **Watch the colors change** - The page background shifts through the full color spectrum
4. **Go clockwise or anticlockwise** - The color moves back and forth based on your drag direction

## Features

- ✅ **Donut chart design** - Highly transparent color wheel overlay
- ✅ **Opaque page background** - Shows selected color clearly
- ✅ **Touch & mouse support** - Works on desktop and mobile
- ✅ **Smooth transitions** - No jumpy color changes
- ✅ **Visual feedback** - Shows current hue degree and HSL value
- ✅ **Responsive design** - Adapts to any screen size
- ✅ **High DPI support** - Sharp rendering on retina displays

## Technical Details

- **Pure HTML/CSS/JS** - No dependencies
- **Canvas rendering** - Smooth, performant graphics
- **HSL color space** - Easy hue manipulation (0-360°)
- **Gesture recognition** - Detects clockwise/anticlockwise movement
- **Touch-friendly** - Works with thumb swipes on mobile
